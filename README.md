Java JUnit – CPF Validator | CHECKPOINT 01 

Projeto desenvolvido em Java para validar números de CPF utilizando testes unitários com JUnit 5.
Este projeto foi criado como parte de um desafio acadêmico para praticar Testes Unitários, garantindo que os métodos da classe fornecida funcionem corretamente.

📚 Objetivo

Validar o comportamento da classe CodigoPessoaFisica, verificando:
- Se um CPF válido é aceito
- Se um CPF inválido é rejeitado
- Se o método de remoção de caracteres especiais funciona corretamente
- Os testes foram implementados utilizando JUnit 5 e organizados em uma Test Suite.

🧪 Testes implementados

A classe de testes verifica três cenários principais:
- ✔ Validação de CPF válido
- ✔ Validação de CPF inválido
- ✔ Remoção de caracteres especiais (. e -)

⚙️ Tecnologias utilizadas

- Java
- JUnit 5
- Eclipse IDE
- Git
- GitHub

▶️ Como executar os testes

Abra o projeto em uma IDE Java (ex: Eclipse ou IntelliJ)
Localize a classe:
CodigoPessoaFisicaTeste.java
Execute como:
Run As → JUnit Test
ou execute a suíte:
SuiteTeste.java
